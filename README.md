
# JavaScript PDoc parser

An attempt to parse [PDoc](http://pdoc.org/syntax.html) with [node.js](http://nodejs.org)

## How to Install

Clone and have fun

## How to use

Go to your project's root directory and run

    ndoc
    open file://./docs/index.html
